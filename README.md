## BELAJAR GIT 

1. membuat inisialisasi repo di local

        git init

2. cara commit
        
        git commit -m "pesan"

3. melihat status di local

        git status

4. melihat log file

        git log
        git log --oneline

5 melihat perbedaan antar commit

        git diff
        git diff versiId1..versiId2
